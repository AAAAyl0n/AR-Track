# ARTrack

![效果图](D:\02-Projects\ARTrack-main\5.Docs\效果图.png)

ARTrack是一个非机动车AR抬头显示（HUD）。它设计灵感来源于飞机驾驶舱的HUD系统，旨在将关键信息直观地展示在骑行用户的视线范围内，从而减少分心、提高驾驶安全性，并提供一些个性化的设置。它允许骑手在保持对周围环境的高度关注的同时，快速获取速度、导航和通信数据。
